# EXAMPLE PROCESS MOTION DATA CALL
python processmotiondata.py -xyhm -e /path_to_folder/scheduled-events -c /path_to_folder/pre-processed.csv -d /path_to_folder/zebrafish-tracker.cells -g /path_to_folder/box2drug27_dmso_vs_box2drug27_drug_scripted_inputgenotypeids -s /path_to_folder/sectionsfile -j /path_to_folder/PlotParameters -r /path_to_folder/zebrafish-tracker.cells -m path_to_folder 

## parameters
-xhym -> will create position based heatmaps \
-e -> same scheduled events used to schedule stimuli for run \
-c -> centroids file, csv containing x, y positions of the fish in each ROI \
-d -> dpix file, but should be given cells file, generated by GUI \
-g -> genotype ids, shown below \
-s -> sections file, shown below \
-j -> parameters for plotting \
-r -> cells file, generated by GUI \
-m -> path to experiment folder 

make sure that both of below correspond to the number of ROIs / times of the videos - if a run ends early, you will have to change the sectionsfile and the scheduled-events file

other_name_of_group:10,11,12,13 ...
